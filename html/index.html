About

Welcome to ircbot.c official documentation. ircbot.c is a simple library (ircbot) that provides easy to use functions providing basic necessities for an IRC bot, such as sending, receiving and custom commands.

To learn more, see public API reference.

There is an included utility library, ircdebug. Documentation for it is available here.

To see what's planned, take a look at the TODO page.

Example usage

See the Chatter IRC bot documentation and source code for practical usage.

Building

Requirements

  • C compiler with C99 standard support (gcc, clang)
  • CMake, version >= 3.0

Steps

The bot should run fine both on Windows and Linux. Building instructions are very simple:

git clone https://github.com/makos/ircbot.c.git
cd ircbot.c
mkdir build && cd build
cmake .. && make ircbot

This will create a library file libircbot.a in build/lib/.
