libdebug - a simple logging library

About

libdebug is a very simple logging library, that can be used in place of debugging and logging printf() statements. It provides functions to enable and disable logging programatically, so you can disable all logging with one statement without having to comment out or delete blocks of code.

API Reference

  • File debug.h
    • debug_log()
    • debug_set_out()
    • debug_enable()
    • debug_disable()
    • debug_is_enabled()
    • debug_get_fd_str()

Building

Building instructions are similar to libbot's.

git clone https://github.com/makos/ircbot.c.git
cd ircbot.c
mkdir build && cd build
cmake .. && make libdebug

This will create a library file libdebug.a in build/lib/.
