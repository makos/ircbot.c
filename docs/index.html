ircbot.c Documentation

About

Welcome to ircbot.c official documentation. ircbot.c is a simple library (libbot) that provides easy to use functions providing basic necessities for an IRC bot, such as sending , receiving , joining and leaving channels and custom commands .

To learn more, see public API documentation.

To see what's planned, take a look at the TODO page.

Building

Requirements

  • C compiler with C99 standard support (gcc, clang)
  • CMake, version >= 3.0

Steps

The bot should run fine both on Windows and Linux. Building instructions are very simple:

git clone https://github.com/makos/ircbot.c.git
cd ircbot.c
mkdir build && cd build
cmake .. && make bot

The library will be located in the build/lib directory.

Public API

Public API functions are contained in the bot.h file. See that file's documentation to learn the API.

The file commands.h contains basic commands to showcase usage of the command system.
